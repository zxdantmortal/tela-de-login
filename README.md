🚀 Tecnologias
Este projeto foi construído utilizando as seguintes ferramentas:

React Native - Framework para desenvolvimento cross-platform.

Expo - Plataforma de ferramentas e serviços para aplicações React.

TypeScript - Superset de JavaScript que adiciona tipagem estática.

React Navigation - Roteamento e navegação para apps mobile.

📦 Instalação e Uso
Para clonar e executar esta aplicação, você precisará do Git, Node.js e um gerenciador de pacotes (npm, yarn ou pnpm) instalados em seu computador.

1. Clone o repositório
Bash
git clone https://github.com/seu-usuario/dock.git
cd dock
2. Instale as dependências
Bash
npm install
# ou
yarn install
3. Inicie o servidor do Expo
Bash
npx expo start
Após o comando acima, você pode visualizar o app de três formas:

Usando o app Expo Go no seu celular físico (escaneando o QR Code).

Pressionando a para abrir no emulador Android.

Pressionando i para abrir no simulador iOS.

🛠️ Funcionalidades principais
✅ Dashboard intuitivo: Visualização rápida do status do sistema.

✅ Sincronização em tempo real: Dados sempre atualizados.

✅ Modo Offline: Suporte para uso sem conexão com a internet.

✅ Notificações Push: Alertas importantes diretamente no dispositivo.
